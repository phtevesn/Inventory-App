fastapi → the web framework

uvicorn[standard] → app server + nice extras (uvloop, httptools, websockets)

sqlalchemy → ORM

pydantic → data validation

PyJWT → JWT tokens

passlib[bcrypt] + bcrypt → password hashing

pydantic-settings → load .env or environment config

psycopg[binary] or mysqlclient → DB driver (if needed)

pip install fastapi 'uvicorn[standard]', sqlalchemy, pydantic, pyjwt, "passlib[bcrypt]", pydantic-settings, psycopg2-binary, python-multipart
